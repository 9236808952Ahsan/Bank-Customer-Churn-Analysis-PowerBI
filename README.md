# Bank-Customer-Churn-Analysis-PowerBI

🏦 Bank Customer Churn Analysis
📊 Project Overview
This project analyzes bank customer data to identify key factors contributing to customer churn. It helps the bank understand which segments are at risk and how to improve retention.

🖼️ Dashboard Preview
🛠️ Tools & Technologies Used
Power BI Desktop: For Data Visualization & DAX.

Power Query: For Data Cleaning & Transformation.

Excel/CSV: As the data source.

📈 Key Insights Found
High Risk Age Group: Customers aged 45-55 show the highest churn rate.

Geographic Impact: Germany has a significantly higher churn (~32%) compared to France and Spain.

Credit Score Factor: Low credit score bins show a direct correlation with customer exits.
